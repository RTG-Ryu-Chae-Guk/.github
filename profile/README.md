## 📝 소개
교통·상권·도시 데이터를 활용한 상권 분석 플랫

기능(간단)
- 사업자 전용 익명게시망
- 지도
- AI분석
- 상권 상세 데이터 조회
- 문서화 및 다운로드
- 실시간 기반 시각화 탭

<br />
사용 데이터
<br />
1.서울시 실시간 도시 데이터(서울 열린데이터 광장)
<br />
2.서울시 상권분석 데이터
<br />
3.국가교통정보센터(CCTV화상자료)
<br />
4.국세청_사업자등록정보 진위확인 및 상태조회 서비스
<br />
5.국토교통부_상업업무용 부동산 매매 실거래가 자료
